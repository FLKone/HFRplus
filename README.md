Utilisation
-------------------------

Un fichier Config.h est nécessaire (dans le répertoire Classes) :

``` objective-c
//
//  Classes/Config.h
//

#define kTestFlightAPI                  @"TestFlight API Key"
#define kGoogleAnalyticsAPI             @"Google Analytics API Key"
```
