Que faire ?

À terme, les deux applications restantes seront enfin (:o) différentes :

  • HFR+ pour assurer la compatibilité avec les anciennes versions d'iOS.

Il est indispensable de migrer vers l'une ou l'autre. Aucune mise à jour ne sera possible après la suppression definitive de l'App Store.
